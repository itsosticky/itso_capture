# Itsosticky screen capture - Chrome extension

A simple Chrome extension for easy screen captures.

[![View demo on Youtube](https://i.itsosticky.com/18ce68n.png)](https://www.youtube.com/watch?v=ilKU8eoWr6c "View demo on Youtube")

[Itsosticky screen capture extension on Chrome webstore](https://chrome.google.com/webstore/detail/itsosticky-screen-capture/igaopnngccagbhclcohemceikfoeeaco)

A simple Google Chrome extension for [https://itsosticky.com/](https://itsosticky.com/).

Drag a region to capture.

Itsosticky is still in a beta stage and doesn't yet have a fully open API.

Google intend to [retire their Chrome apps platform](https://blog.chromium.org/2016/08/from-chrome-apps-to-web.html)
for non-Chrome-OS operating systems in Novemeber 2016 and this extension is released
in an early stage to better support these systems over the coming year.
